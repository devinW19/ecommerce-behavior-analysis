# ecommerce-behavior-analysis
An end-to-end data analysis project using the Online Retail II dataset to perform RFM (Recency, Frequency, Monetary) segmentation and identify high-value customer cohorts.

The Problem: Raw e-commerce data is often noisy, containing cancellations and missing customer data, making it difficult to identify loyal customers.

The Solution: I developed a Python-based pipeline to clean 1.04 million rows of data, engineered RFM features, and segmented customers into actionable categories (e.g., Champions, At Risk).

Language: Python

Libraries: Pandas, NumPy, Matplotlib, Seaborn

Tools: Jupyter Notebooks, Git/GitHub
